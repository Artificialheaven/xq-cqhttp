# 事件信息

## 事件上报

### 获取登陆QQ信息

| 字段名 | 数据类型 | 可能的值 | 说明 |
| ----- | ----- | ----- | ----- |
| action | str | get_login_info | 侦测信息类型 |
| - | - | - | 原先CQHTTP后面还有一段，现在不需要了 |

| 回传字段名 | 数据类型 | 可能的值 | 说明 |
| ----- | ----- | ----- | ----- |
| user_id | num | - | 登陆的姬气人QQ |
| nickname | str | 0 | 登陆的姬气人QQ名 |

### 其余事件

不处理。抛出异常后返回ERROR（不确定）。
